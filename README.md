# Rorahi

A work-in-progress, 3D surface graph in WebGL

## Run An Example

Start the dev server

    npm run server

Then view an example:

- http://localhost:8088/examples/simple/
- http://localhost:8088/examples/auckland/
- http://localhost:8088/examples/kitchensink/
